# ModuOS Network Manager

The official **Network Manager** component for the ModuOS operating system.

---

## ⚠️ Prerequisites

To successfully build and deploy this project, ensure you have the following environments configured:

* **ModuOS Docker Container:** Required to provide the necessary compilation environment and dependencies.
* **ModuOS Source Code:** Required to correctly link and package the final `.sqr` files into the system ISO image.

## 🛠️ Build Process

1. Ensure the **ModuOS Docker container** is running.
2. Link your **ModuOS source code** directory to the project environment.
3. Execute the build command to generate the `.sqr` files.
4. Integrate the resulting files into your ModuOS ISO build pipeline.
